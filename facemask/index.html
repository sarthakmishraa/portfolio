Face Mask Detector

Author

 

Sarthak Mishra

This project does face mask detection using Computer Vision. As now a days mask is a powerful way to prevent us from SARS-CoV2. So everyone should wear mask all time. To check this I made a project. This model can be deployed then used at entrance to check if person is wearing mask or not. I trained the model in 2 chunks, one with mask and second without mask. Association : keras, numpy, cv2, datetime.
